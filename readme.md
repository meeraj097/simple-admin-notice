# Simple Admin Notice Manager

A lightweight WordPress plugin that displays custom, dismissible admin notices in the WordPress dashboard.

## Features
- Displays admin dashboard notices
- Dismissible notices using JavaScript
- Clean plugin structure

## Tech Stack
- PHP
- JavaScript
- WordPress Hooks

## Installation
1. Clone this repository
2. Upload the folder to `/wp-content/plugins/`
3. Activate the plugin from the WordPress dashboard
